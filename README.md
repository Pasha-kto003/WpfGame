# WpfGame

## Описание игры
Данная игра была реализована в качестве лабораторной работы по дисциплине "Методы и технологии продвижения информационных ресурсов". Для реализации использовались такие инстурменты как WPF + C#.
Игра WpfGame является копией старой игры Space Invaders она простая и состоит из 4х уровней. Первые три уровная это уровни с врагами, на каждом уровне количество врагов увеличивыается, а также увеличивается их агрессия по отношению к игроку. На четвертом уровне уже появляется босс игры, которого нужно победить путем попадания нескольких снарядов по нему
Далее идут визуальные примеры примеры уровней

### Управление
A или стрелка влево для движения влево
D или стрелка вправо для движения вправо
Enter и только пробел для стрельбы из орудия
